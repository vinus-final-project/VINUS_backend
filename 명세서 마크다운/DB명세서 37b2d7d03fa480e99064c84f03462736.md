SessionStatus = ACTIVE, COMPLETED, EXPIRED, CANCELLED

```jsx
from enum import Enum

class SessionStatus(str, Enum):
    ACTIVE = "ACTIVE" // 세션 진행 중(ex 주문 진행 중)
    COMPLETED = "COMPLETED" // 세션 정상 종료(ex 주문 완료 등)
    EXPIRED = "EXPIRED" // 세션 만료(180초)
    CANCELED = "CANCELED" // 비정상적인 세션 강제 취소
```

```jsx
se_status = Column(
    Enum(SessionStatus),
    nullable=False,
    default=SessionStatus.ACTIVE
)
```

레그용 문서 만들어주는 함수

```jsx
def build_menu_document(menu):
    return f"""
메뉴명: {menu.name}

카테고리: {menu.category}

설명:
{menu.description}

가격:
{menu.price}원

원재료:
{", ".join(menu.ingredients)}

알러지:
{", ".join(menu.allergies)}

선택 가능 옵션:
{", ".join(menu.options)}
"""
```

DB관련 함수 명세서

MenuRepository

| Method | 입력 | 반환 | 설명 |
| --- | --- | --- | --- |
| find_by_id | menu_id | Menu | 메뉴 ID로 메뉴 조회 |
| find_all | 없음 | List<Menu> | 전체 메뉴 조회 |
| find_by_category | category | List<Menu> | 카테고리별 메뉴 조회 |
| exists | menu_id | Boolean | 메뉴 존재 여부 확인 |

> **사용하는 Controller**
> 
- OrderController.create_order_item()
- RecommendationController.recommend_menu()
- SystemController.get_menu_info()
