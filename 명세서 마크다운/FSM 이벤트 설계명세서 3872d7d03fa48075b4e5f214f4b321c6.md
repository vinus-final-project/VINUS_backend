# FSM 이벤트  설계명세서

### FSM Satus

| fsm_state | order_item.status | Android 화면 |
| --- | --- | --- |
| INIT | - | 메인 |
| ORDERING | SELECTING_REQUIRED_OPTION | 필수 옵션 |
| ORDERING | ASKING_OPTIONAL_OPTION | 선택 옵션 / 수량 |
| ORDERING | order_item == null | 장바구니 |
| PAYMENT | - | 결제 |
| COMPLETE | - | 완료 |

### 총 이벤트 목록

```python
SELECT_ORDER_TYPE

SELECT_MENU
SET_QUANTITY
CANCEL_ORDER_ITEM

SELECT_REQUIRED_OPTION
SELECT_OPTIONAL_OPTION
SKIP_OPTIONAL_OPTION

SHOW_CART

INCREASE_CART_ITEM

REQUEST_RECOMMENDATION
ACCEPT_RECOMMENDATION

REQUEST_MENU_INFO

START_PAYMENT
PAYMENT_SUCCESS
PAYMENT_CANCEL

CANCEL_SESSION
EXPIRE_SESSION
```

| Event | 현재 상태 | 설명 | 리턴 | 조건 | 실행 Method | 실패시 코드 | Session 변경 | 다음 상태 |

| --- | --- | --- | --- | --- | --- | --- | --- | --- |

| CANCEL_ORDER_ITEM | ORDERING | 현재 선택 메뉴 취소 | 없음 | ① fsm_state==ORDERING ② order_item 존재 | OrderController.cancel_order_item() | INVALID_STATE, ORDER_ITEM_NOT_FOUND | session.order_item 제거 | ORDERING |

OrderItem 완료 시 검증

| 순서 | 조건 | 실패 코드 |
