# 메모리 명세서

## 세션 메모리

```
Session
├─ session_id          # UUID
├─ fsm_state           # 현재 FSM 상태(INIT, ORDERING, PAYMENT, COMPLETE)
├─ order_type          # 매장(STORE) / 포장(TAKEOUT)
├─ order_item          # 현재 작성 중인 주문 항목(Pending Item)
├─ cart                # 주문 완료된 메뉴 목록
├─ recommendation_list # 추천 메뉴 리스트
└─ logs                # 세션 종료 시 DB에 저장될 로그 버퍼(리스트)
└─ message (또는 tts)
```

- ex)
    
    ```python
    {
      "session_id": "UUID",
      "created_at": "2026-06-23T10:00:00",
    
      "fsm_state": "ORDERING",
      "order_type": "STORE",
    
      "order_item": {
        "menu_id": 1,
        "quantity": 1,
        "selected_options": {
          "size": "LARGE"
        },
        "status": "SELECTING_REQUIRED_OPTION"
      },
    
      "cart": [],
    
      "recommendation_list": 15,16
    
      "logs": []
    }
    ```
    

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| session_id | str | UUID |
| created_at | datetime | 세션 생성 시간 |
| fsm_state | FSMState | 현재 FSM 상태 |
| order_type | OrderType | None | 매장/포장 여부 |
| order_item | OrderItem | None | 현재 작성 중인 주문 항목 |
| cart | List | 장바구니 |
| recommendation_list | List<Menu> | 추천 메뉴 리스트(menu_id) |
| logs | List | 세션 로그 버퍼 |

### 로그 메모리

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| speaker | SpeakerType | 발화 주체(USER, AI, SYSTEM) |
| message | str | 발화 내용 |
| intent | str | None | 분석된 의도 |
| created_at | datetime | 생성 시간 |

오더 타입 정의

| 값 | 설명 |
| --- | --- |
| STORE | 매장 |
| TAKEOUT | 포장 |

### 카트 저장 규칙

| 조건 | 처리 |
| --- | --- |
| 동일 메뉴 + 동일 옵션 | quantity 증가 |
| 동일 메뉴 + 다른 옵션 | 신규 CartItem 생성 |
| order_item.status = COMPLETE | cart 추가 가능 |
| order_item.status ≠ COMPLETE | cart 추가 불가 |

```python
for item in cart:

    if (
        item.menu_id == pending_item.menu_id
        and item.selected_options == pending_item.selected_options
    ):
        item.quantity += pending_item.quantity
        return

cart.append(pending_item)
```

## 오더 아이템 메모리

```python
order_item
├─ menu_id
├─ quantity # 수량
├─ selected_options # 선택된 옵션들
└─ status # 주문 중인 메뉴 상태

order_item의 status
├─ SELECTING_REQUIRED_OPTION # 필수 옵션 선택 중
├─ SELECTING_QUANTITY        # 수량 선택 중
├─ ASKING_OPTIONAL_OPTION    # 추가 옵션 선택 여부 확인 중
└─ COMPLETE                  # 주문 항목 작성 완료
```

- ex)
    
    ```python
    # 아메리카노 주세요
    {
      "menu_id": 1,
      "quantity": 1,
      "selected_options": {},
      "status": "SELECTING_REQUIRED_OPTION"
    }
    ```
    
    ```python
    # 라지 아이스로 주세요
    		{
      "menu_id": 1,
      "quantity": 1,
      "selected_options": {
        "size": "LARGE",
        "temperature": "ICE"
      },
      "status": "ASKING_OPTIONAL_OPTION"
    }
    # 추가 옵션을 선택하시겠습니까?
    ```
    
    ```python
    # 아니요
    {
      "menu_id": 1,
      "quantity": 1,
      "selected_options": {
        "size": "LARGE",
        "temperature": "ICE"
      },
      "status": "COMPLETE"
    } 
    ```
    
    ```python
    class OrderItemStatus(Enum):
    		SELECTING_REQUIRED_OPTION = 'SELECTING_REQUIRED_OPTION'
    		ASKING_OPTIONAL_OPTION = 'ASKING_OPTIONAL_OPTION'
    		COMPLETE = 'COMPLETE'
    ```
    

| 필드명 | 타입 | 설명 |
| --- | --- | --- |
| menu_id | int | 메뉴 ID |
| quantity | int | 주문 수량 |
| selected_options | dict | 선택된 옵션 목록 |
| status | OrderItemStatus | 주문 작성 상태 |

order_item status ENUM

| 상태 | 설명 |
| --- | --- |
| SELECTING_REQUIRED_OPTION | 필수 옵션 선택 중 |
| SELECTING_QUANTITY | 수량 선택 |
| ASKING_OPTIONAL_OPTION | 추가 옵션 선택 여부 확인 중 |
| COMPLETE | 주문 항목 작성 완료 |
