# DTO 명세서

| DTO | 방향 | 용도 |
| --- | --- | --- |
| VoiceRequest | Android → Server | 음성 전달 |
| PaymentResultRequest | 결제모듈 → Server | 결제 완료/취소 전달 |
| SessionResponse | Server → Android | 일반 응답 |
| ErrorResponse | Server → Android | 오류 응답 |

VoiceRequest

Android → STT

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| audio | Byte[] (PCM) | O | VAD가 분리한 PCM 음성 데이터 |
| sample_rate | Integer | O | 샘플링 레이트(16000Hz) |
| channels | Integer | O | 채널 수(1=Mono) |
| timestamp | DateTime | X | 요청 시간(로그용) |

STTResult

STT → RapidFuzz

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| text | String | O | STT 변환 결과 |

NormalizeResult

RapidFuzz → Rule Parser

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| text | String | O | 보정된 텍스트 |

ParseResult

Rule Parser → Rule Engine

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| intent | Intent | O | 추출된 Intent |
| entities | Map<String,Object> | O | 추출된 Entity |
| source | RULE / LLM | O | 파싱 출처 |

EventList

Rule Engine → Event Executor

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| events | List | O | 실행할 Event 목록 |

SessionResponse

Event Executor → Android

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| response_type | ResponseType | O | 응답 종류 |
| session_id | UUID | O | 세션 식별자 |
| success | Boolean | O | 처리 성공 여부 |
| message | String | O | Rule Engine 또는 LLM이 생성한 사용자 안내 문구(TTS 출력 및 UI 표시용) |
| fsm_state | FSMState | O | 현재 FSM 상태 |
| pending_event | FSMEvent | X | 다음 입력 요청(Event) |
| cart | List | X | 현재 장바구니 |
| recommendation_list  | recommendation_list  | X | 추천 메뉴 |
| error_code | ErrorCode | X | 오류 코드 |
| session_end | Boolean | O | 세션 종료 여부 |

ㄴResponseType

| 값 | 설명 |
| --- | --- |
| NORMAL | 일반 응답 |
| ERROR | 오류 응답 |
| PAYMENT_SUCCESS | 결제 완료 |
| PAYMENT_CANCEL | 결제 취소 |
| SESSION_END | 세션 종료 |

PaymentResultRequest

결제 모듈 → Backend

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| payment_result | PaymentStatus | O | SUCCESS / FAIL / CANCEL |
| payment_id | String | O | 결제 번호 |

ㄴPaymentStatus

| 값 | 설명 |
| --- | --- |
| SUCCESS | 결제 성공 |
| FAIL | 결제 실패 |
| CANCEL | 결제 취소 |

ErrorResponse

Backend → Android

| 필드 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| session_id | UUID | O | 세션 식별자 |
| error_code | ErrorCode | O | 오류 코드 |
| message | String | O | 사용자 안내 문구 |
| session_end | Boolean | O | 세션 종료 여부 |
| fsm_state | FSMState | O | 오류 발생 당시 상태 |

DTO 흐름

| From | DTO | To |
| --- | --- | --- |
| Android | VoiceRequest | STT |
| STT | STTResult | RapidFuzz |
| RapidFuzz | NormalizeResult | Rule Parser |
| Rule Parser | ParseResult | Rule Engine |
| Rule Engine | EventList | Event Executor |
| 결제 모듈 | PaymentResultRequest | FSM |
| Event Executor | SessionResponse | Android |
| Event Executor | ErrorResponse | Android |
